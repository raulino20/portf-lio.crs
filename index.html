HTML (HYPER TEXT MARKUP LANGUAGE)

o HTML é o componente base da web. Isso quer dizer que ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos; por lá voçê tambem pode adicionar arquivos externos, como o CSS (style.css), linkando-o diretamente na pagina

Código uso
H1 : usado para Titulo em sites
STRONG : usado para deixar frases em Negrito
P : usado para paragrafos, montar textos
IMG : usado para adicionar imagens ao seu conteúdo

CSS (CASCADING STYLE SHEET)

Trata-se de uma linguagem de marcação, amplamente utilizada com HTML ou XHTML, e representa diversas possibilidades para a formatação. O CSS ajuda a editar, alinhar, remover e trabalhar no espaço entre elementos de uma página.

VEJA ESSES EXEMPLOS
Código uso
Background-color : a cor do fundo do conteuudo será mudado
text-align : o texto será alinhado onde voçê quiser ( left, right...)
margin : voçê pode mudar a margin para distancia-los
align-items: o conteudo será posicionado onde voçê quiser ( left, right...)
